
Itefu は、RGSS3（RPGツクールVX Aceのゲームエンジン）上で動作する、汎用ゲームフレームワークです。

利用方法は[Five RingsのWebサイト](https://five-rings.neocities.org/inside-itefu/)や、Windows用RPG『Nie』の使われ方を参照してください。
